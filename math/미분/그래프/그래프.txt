일차함수 그래프 : 직선
이차함수 그래프 : 곡선

기울기 : x가 변화할 때, y는 얼마나 빠르게 변하는지
일차식 기울기 : f(x) / x
이차식 기울기 : 순간 변화율 과 평균 변화율을 알아야 함

평균 변화율 : f(y) - f(x) / y - x

h = y - x
순간 변화율 : f(x + h) - f(x) / h
함수에 대입 후 h을 0을 대입해서 나온 수가 순간 변화율

f'(x): 기본 함수 f(x)의 미분 (함수의 순간 변화율)

미분 구하는 공식 : 함수에 차수의 수을 앞으로 가져와 계수와 곱한 후 차수에 -1을 한 후 상수는 날린다

 ex) 4x^3 + 3x^2 - 1 의 f'(x) = 12x^2 + 6x = 순간 변화율(미분)


기울기는 그래프가 해당 지점에서 얼마나 기울어져 있는지 와 어떤 방향으로 가야 가장 가파르게 올라갈 수 있는지를 알려준다. ( 기울기가 + 일땐 오른쪽 기울기가 - 일땐 왼쪽으로 가야 가파르다)


극소점(local minimum) : 기울기가 음수에서 양수로 넘어갈 때 기울기가 0을 찍고 가는 부분
        최소점(global minimum): 극소점 중 제일 낲은 점

극대점(local maximum) : 기울기가 양수에서 음수로 넘어갈 때 기울기가 0을 찍고 가는 부분
        최대점(global maxinum): 극대점 중 제일 높은 점

안장점: 기울기가 양수에서 양수로 넘어갈 때 기울기가 0을 찍고 가는 부분분


편미분 : 함수를 변수 하나에 대해서만 미분
편미분 방식 : 변수 하나를 미분하여 이외 변수들은 상수 취급하여 날려준다. ( 꼬브라진 d : 델)
    ex) 2x^2 + 3y^2 = 4x
                      6y

                      [
                      [4x],
                      [6y]
                      ]
