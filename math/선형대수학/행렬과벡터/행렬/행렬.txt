행렬(Matrix) : 수를 직사각형의 형태로 나타낸 것
A = [
    [1, 1, 0, 2], 원소: 1, 1, 0, 2
    [2, 1, 4, 1], 원소: 2, 1, 4, 1
    [0, 3, 2, 1]  원소: 0, 3, 2, 1
     ]

행(row) : 행렬의 가로줄

A의 행:
1행 = 1, 1, 0, 2
2행 = 2, 1, 4, 1
3행 = 0, 3, 2, 1


열(Columns) : 행렬의 세로줄
A의 열:
1열 = 1, 2, 0
2열 = 1, 1, 3
3열 = 0, 4, 2
4열 = 2, 1, 1

A = 3*4 행열

A12 = 1
A24 = 1

행렬은 보통 대문자로 나타낸다.
