결정 트리는 다른 머신러닝 알고리즘 보다 성능이 떨어진다.

하지만 결정 트리를 사용하는 이유는 : 응용하면 성능이 좋은 다른 모델들을 만들 수 있기 때문

대표적인 기법
 1. 앙상블(ensemble: 합주단)
    1) 여러 독립적인 객체들이 만들어내는 조화로운 단체
    2) 수 많은 모델들을 만들고, 이 모델들의 예측을 합쳐서 종합적인 예측을 하는 기법

    종류 : 랜덤 포레스트


랜덤 포레스트 : 트리 모델들을 임의로 많이 만들어서 다수결 투표로 결과를 종합하는 알고리즘
    임의성을 더하는 방법
        1) bootstrapping: 갖고 있는 데이터 셋으로 다른 데이터 셋을 만들어내는 방법
            만드는 이유:
                1) 모든 모델을 정확히 똑같은 데이터 셋으로 학습시키면 결과 다양성 떨어질 수 있음
                2) 이 문제를 예방하기 위해, 각 모델을 임의로 만들어준 bootstrap 데이터 셋으로 학습시켜준다
