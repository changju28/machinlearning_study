결정트리 : 예 / 아니오 질문들 이 있고 그 질문들을 따라가면서 답을 해나가면서 분류 하는 알고리즘

    ex)         안전 벨트를 했나요?
        예                           아니오
       (생존)                        (사망)


                시속 100km가 넘었나요?
       예                           아니오
      (사망)                       (생존)


1. 하나의 시작 질문에서 뻗어나감
2. 내려갈 때마다 예 아니오로 나아간다

root 노드 : 질문
leaf 노드 : 사망 생존과 같은 특정 예측값을 갖는다.

결정트리를 만들때 과제를 만드는법 (지니 불순도)

1. 이미 질문과 분류들이 정해졌었음
2. 원래 학습 데이터를 보면서 각 노드들을 정함

지니 불순도 :
1. 데이터 셋 안에 서로 다른 분류들이 얼만큼 섞여있는지
2. 작을수록 데이터 셋이 순수하고, 클수록 데이터 셋이 불순하다

ex) 독감 데이터 셋

독감      일반 감기
                         독감일 확률    독감이 아닐 확률
지니 불순도 공식 : GI = 1 - p(flu)^2  -  p(not_flu)^2

p(flu) = 독감인 환자 / 전체 환자
p(not flu) = 독감이 아닌 환자 / 전체 환자


노드 중요도

NI = n/m * (GI - n(left)/m) * GI(left) - n(right)/m * GI(right)
